Política de Privacidade

Última atualização: Março 2026

1. Informações Coletadas

O Bob Assistant coleta apenas as informações necessárias para funcionamento da integração com TikTok:

  • Nome de usuário do TikTok
  • ID da conta
  • Tokens de acesso (armazenados de forma segura)

2. Uso das Informações

As informações são utilizadas exclusivamente para:

  • Autenticar via Login Kit do TikTok
  • Postar vídeos na conta do usuário (quando autorizado)
  • Melhorar a experiência do assistente

3. Compartilhamento de Dados

Não compartilhamos suas informações com terceiros. Seus dados são utilizados apenas para as funcionalidades do bot.

4. Segurança

Implementamos medidas de segurança para proteger seus dados contra acesso não autorizado.
